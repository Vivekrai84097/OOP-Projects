# Banking Application (Java OOP Project)

This is a simple Banking Application built using Core Java and Object-Oriented Programming concepts.

## 🚀 Features

- Create Bank Account
- Deposit money
- Withdraw money
- Display account details
- Savings Account with Interest calculation
- Current Account with account type display
- Inheritance and Method Overriding implemented

---

## 🧠 Concepts Used

- Classes and Objects
- Constructors
- Encapsulation (basic level)
- Inheritance
- Method Overriding
- super keyword

---

## 📂 Project Structure

- BankAccount (Parent Class)
- SavingAccount (Child Class)
- CurrentAccount (Child Class)
- BankingApplication (Main Class)

---

## ⚙️ How it works

1. Create a bank account
2. Perform deposit and withdraw operations
3. For Savings Account, interest is added using addInterest()
4. For Current Account, account details are displayed with account type
5. All operations are printed on console

---

## 📌 Example Output

- Deposited Amount: 2500  
- Withdrawn Amount: 1200  
- Interest added successfully  
- Account details displayed
