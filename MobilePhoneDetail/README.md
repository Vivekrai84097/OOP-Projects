# Project 2: Mobile Phone Details (Java OOP)

## Project Overview
This project is a simple Java OOP program that simulates a mobile phone system with basic functionalities like charging and using the phone.

---

## Features
- Brand of phone
- Model name
- Battery percentage tracking
- Charge battery function
- Use phone function (battery decreases)

---

## OOP Concepts Used
- Class and Object
- Constructor
- Methods
- Validation logic

---

## Constructor + Validation
- Constructor is used to initialize brand, model, and battery percentage.
- Validation ensures battery percentage is always between 0 to 100.

---

## Methods
### chargeBattery()
- Increases battery percentage up to 100

### usePhone()
- Decreases battery percentage based on usage

---

## Purpose
To practice Java OOP concepts with real-life simulation and data validation.
