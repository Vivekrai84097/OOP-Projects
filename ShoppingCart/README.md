# Shopping Cart System (Java OOP Project)

## Project Overview
This is a basic Shopping Cart System built using Core Java (OOP concepts) without using ArrayList.

The project demonstrates:
- Classes & Objects
- Constructors
- Encapsulation
- Arrays of Objects
- Methods
- Basic Cart Management

---

## Classes Used

### 1. Product
Stores product details:
- Product ID
- Product Name
- Product Price

#### Features:
- Create product
- Display product details
- Get product ID
- Get product price

---

### 2. Customer
Stores customer details:
- Customer ID
- Customer Name

#### Features:
- Display customer details

---

### 3. Cart
Stores products added by the customer using an array.

#### Features:
- Add product to cart
- View all products
- Remove product by ID
- Calculate total cart price

---

## Key Functionalities

### Add Product
Adds products into the cart until cart capacity is full.

### View Cart
Displays all added products.

### Remove Product
Removes a product using product ID and shifts remaining products.

### Calculate Total
Calculates total price of all products in the cart.
