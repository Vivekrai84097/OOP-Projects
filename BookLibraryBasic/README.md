# Project 3: Book Library Basic (Java OOP)

## Project Overview
This is a basic Java OOP mini project that simulates a simple library system for issuing and returning books.

---

## Features
- Book title
- Author name
- Issue status tracking
- Issue book function
- Return book function
- Display book information

---

## OOP Concepts Used
- Class & Object
- Constructor
- Methods
- Conditional Logic

---

## Constructor Use
- Constructor initializes title, author, and issue status when the book object is created.

---

## Methods
### issueBook()
- Checks if the book is already issued
- Issues the book if available

### returnBook()
- Returns the book if currently issued

### displayInfo()
- Displays complete book details

---

## Purpose
To practice Java OOP concepts using a real-life book library system.
