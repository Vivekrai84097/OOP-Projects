# Vehicle Management System

## Overview
This is a basic Java OOP project based on **Inheritance** and **Method Overriding**.

The project manages different types of vehicles such as:
- Car
- Bike

Each vehicle can:
- Start
- Stop
- Show Fuel Type

---

## Concepts Used
- Class & Object
- Inheritance
- Method Overriding
- Runtime Polymorphism
- Packages

---

## Class Structure

### Parent Class
`Vehicle`

Methods:
- `start()`
- `stop()`
- `fuelType()`

### Child Classes
1. `Car`
2. `Bike`

Both child classes override the methods of the parent class.
