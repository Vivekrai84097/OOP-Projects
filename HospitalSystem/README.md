# Hospital System

## Overview
This is a Java OOP project based on **Abstraction** and **Method Overriding**.

The system represents different types of patients in a hospital and their treatment details.

Patient types:
- Indoor Patient
- Outdoor Patient

---

## Concepts Used
- Abstract Class
- Method Overriding
- Inheritance
- Runtime Polymorphism

---

## Class Structure

### Abstract Class
`Patient`

Method:
- `treatmentDetails()`

---

### Child Classes

#### 1. IndoorPatient
- Patient is admitted in hospital
- Room + treatment included

#### 2. OutdoorPatient
- Patient visits OPD
- Only consultation provided

---

## Sample Output

```text
Indoor Patient : Admitted, Room + Treatment included
Outdoor Patient : OPD Consultation only
```

## Learning Purpose
This project helps to understand:
- Abstraction in Java
- Real-world hospital system modeling
- Method overriding
- Runtime polymorphism
