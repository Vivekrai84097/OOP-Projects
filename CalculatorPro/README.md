# Calculator Pro

## Overview
This is a basic Java OOP project based on **Method Overloading**.

The project performs addition using different types and numbers of parameters.

Features:
- Addition of two integers
- Addition of two double values
- Addition of three integers

---

## Concepts Used
- Class & Object
- Method Overloading
- Polymorphism
- Packages

---

## Class Structure

### Class
`Calculator`

Methods:
- `add(int a, int b)`
- `add(double num1, double num2)`
- `add(int a, int b, int c)`

Each method performs addition with different parameters.

---

## Sample Output

```text
Add : 30
Add : 31.0
Add : 6
```
