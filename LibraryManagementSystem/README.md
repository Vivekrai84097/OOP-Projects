# Library Management System

## Overview
This is a Java OOP project that simulates a simple Library Management System.

The project allows:
- Adding books
- Displaying books
- Issuing books
- Returning books
- Searching books by ID
- Removing books by ID

This project is built using core Java concepts without using ArrayList.

---

## Concepts Used
- Encapsulation
- Classes & Objects
- Arrays
- Object Interaction
- Constructors
- Method Creation
- Searching Logic
- Array Element Shifting

---

## Classes Used

### 1. Book
Stores book details:
- Book ID
- Title
- Author
- Issue status

Methods:
- `issueBook()`
- `returnBook()`
- `displayBook()`

---

### 2. User
Stores user details:
- User ID
- Name

Method:
- `displayUser()`

---

### 3. Library
Manages all library operations.

Features:
- Add books
- Show books
- Issue books
- Return books
- Search books by ID
- Remove books by ID
